## CSCE 312 — Computer Organization

Built a complete computer system from logic gates to assembler using the Nand2Tetris framework.

### Topics Covered
- Digital logic and basic building blocks (gates, combinational and sequential circuits)
- Processor architecture: instruction sets, data paths, and control units
- Memory hierarchy, caching, and performance considerations
- Input/output organization and simple pipelining


### Languages and Tool(s)
- Hardware Description Language (HDL)
- Java Virtual Machine
- Nand2Tetris Software Suite
